# cat-browser

基于cat的浏览器使用情况监控系统

## 使用

-
- git remote rm origin
- git remote add origin {你的仓库地址}  例如：git remote add origin git@code.dianpingoa.com:pc-trade-f2e/apollo-template-static.git
- git fetch
- git pull origin master

- 全局安装smartDoc 用于生成文档

## Command

```
	#测试	
	npm run test	
	#打包	
	npm run build	
	#例子演示	
	npm run demo	
```
